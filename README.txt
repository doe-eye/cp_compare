TM2-xaseco2-plugin for showing local sector-times

INSTALLATION:
1) put the cp_compare.php-file in the xaseco2/plugins-folder
2) in the xaseco2/plugins.xml add the line:
   <plugin>plugin.cp_compare.php</plugin>
3) put the cp_compare.xml-file in the xaseco2-folder (you can adapt some style-things in this file easily)
4) restart xaseco2
